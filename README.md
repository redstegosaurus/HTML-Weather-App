This JavaScript application fetches real-time weather data for a user-specified city using the OpenWeatherMap API. When the user enters a city name, the getWeather() function validates the input and sends a fetch request with an API key. Upon receiving the JSON response, the displayWeather() function processes the data: it converts the temperature from Kelvin to Celsius, extracts the weather description, and dynamically updates the HTML to show the city name, temperature, description, and a high-resolution weather icon. It also includes error handling to alert the user if the city is not found or if the API request fails.
